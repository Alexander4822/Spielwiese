# BAKE_NAVMESH (Unity 6) – 5 Klicks

1. Öffne `Assets/Scenes/Scene_Mission1.unity`.
2. Klicke **Window > AI > Navigation**.
3. Im Tab **Object**: Wähle den Boden `Ground` und setze **Navigation Static** + **Walkable**.
4. Im Tab **Bake**: belasse Standardwerte (oder Agent Radius 0.5, Height 2.0).
5. Klicke **Bake**.

Danach können alle `NavMeshAgent`-Units direkt laufen.
